Competition entry.

- **F**ramework of **O**perating **S**ystem **D**evelopment
- A framework that provides essential build environments and core functionalities required for operating system development

- **Based on ASM and C/C++**
- Currently implements a basic output system (able to print simple messages)

- Designed with architecture extensibility in mind
- Build system customization via Makefile and configuration files
