어린왕자는 모험을 떠나기 위한 비상식량을 마련하기 위해 소행성 B612에 사과나무를 심었다. 사과나무의 높이는 10m이며, 모든 사과 열매는 나무의 꼭대기(높이 10m)에서 열린다고 가정한다.

사과 열매는 생성되는 즉시 낙하를 시작하며, 낙하 속력은 하루에 20m이다. 따라서 열매가 땅에 도달하는 데 걸리는 시간은 12시간이다. 어린왕자는 사과가 땅에 도달하는 순간 즉시 주울 수 있다.

사과가 생성되기까지는 다음과 같은 과정을 거친다.

1. 하나의 가지에서 봉우리가 생기는 데 1시간이 걸리며,
2. 봉우리가 사과 열매로 자라는 데 추가로 4시간이 걸린다. (즉, 가지가 생긴 후 사과 열매가 되기까지 총 5시간이 필요하다.)
3. 열매가 떨어지면, 해당 열매가 달려 있던 가지는 사라진다.

또한, 사과 열매 하나가 땅에 도달하는 순간, 나무에는 새로운 봉우리 1개와 새로운 가지 1개가 동시에 생겨난다.

초기 상태에서 나무에는 가지 1개와 봉우리 1개가 존재한다. 그리고 1시간 후에는 추가로 가지 1개가 더 생겨난다.

이때, 어린왕자가 총 n개의 사과를 줍기까지 걸리는 시간을 구하여라.

---

## 입력

첫째 줄에 테스트 케이스의 개수 T가 주어진다.

둘째 줄부터 T개의 줄에 주어야할 사과의 개수 n이 주어진다.

## 출력

테스트 케이스 T개에 대해 한 줄에 하나씩 총 n개의 사과를 줍기까지 걸리는 시간을 출력한다.

# 풀이

```cpp
#include <iostream>
#include <vector>

int main() {
    int T;
    std::cin >> T;

    for(int i = 0; i < T; i++) {
        int n;
        std::cin >> n;
        std::vector<int> apple;
        apple.push_back(0);
        int sum = 0, time = 0;

        while(sum < n) {
            time++;
            for(int i=sum; i < apple.size(); i++) {
                apple[i]++;
                // 16시간 소요
                if(apple[i] == 16) {
                    sum++;
                    apple.push_back(-1); // 봉우리
                    apple.push_back(-2); // 가지
                }
            }
            if(time == 1 || time == 2) apple.insert(apple.begin() + time, 0);
        }

        std::cout << time << std::endl;
    }
    return 0;
}
```
